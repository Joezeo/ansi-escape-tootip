The plugin for `anis escape code sequence`. 
<br/>
<h3> Features </h3>
<p>Highlight and tooltip the parsed anis escape code sequence from the text on the editor.</p>
<p>Currently supported files:</p>
<ul>
<li>.java</li>
<li>.log</li>
<li>.txt</li>
</ul>
<h3> Settings </h3>
File > Settings > Tool > Anis Escape Tooltip
